About Cloud Nine

Cloud Nine is a weather website powered by OpenWeather's free api.

Capstone Project

Cloud Nine was created by Hoo Wing Wah as his capstone project to overcome inaccuracies and inconsistencies of existing weather websites. Cloud Nine provides a clean user-interface with no bright colours and simple UI designs that make it interpretable for anyone using the website. Through Cloud Nine, Hoo Wing Wah seeks to redefine the way users interact with weather data, providing a platform that is not only informative but also visually appealing and user-centric.
